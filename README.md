Các bước cần thực hiện khi tải code từ github về:
Tóm tắt: Clone về → composer install → tạo .env → chạy
Bước 1: Clone:
- Mở VS Code
- Ctrl + Shift + P
- Gõ Git: Clone
- Dán link GitHub
- Chọn thư mục lưu project
- Bấm Open

Bước 2: ## Setup project
1. composer install
2. cp .env.example .env
3. config database
